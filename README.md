### Issues and contributing

If you would like to create a new __Issue__ or __PR__, please do so in:

- __this repo__ if it's about __color__ changes, like in `colors.less` or `syntax-variables.less`.
- __[DuoTone master template](https://github.com/angelbotto/duotone-syntax)__ if it's about __everything else__, like in improving language support. Changes will trickle down into this theme too.
